# Rock Paper Scissors Game

A classic Rock Paper Scissors game implementation running in the browser console.

## Description

This project is a simple implementation of the Rock Paper Scissors game using clean JavaScript logic. The game is played against the computer in a "Best of 5" format.

## Features

-   **Human vs Computer Logic**: Play against a randomized computer opponent.
-   **Score Tracking**: Keeps track of human and computer scores across rounds.
-   **Console Interface**: Game output and interaction happen directly in the browser developer console.
-   **Best of 5**: The game loops for 5 rounds to determine a winner.

## How to Play

1.  Clone or download this repository.
2.  Open `index.html` in your modern web browser.
3.  Open the **Developer Console** (usually F12 or right-click -> Inspect -> Console).
4.  When prompted, type your choice: `rock`, `paper`, or `scissors`.
5.  View the results of each round and the final score in the console.

## Technologies Used

-   **HTML5**: For loading the script.
-   **JavaScript**: For game logic and interaction.
